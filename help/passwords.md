# Passwords (Vault) Help

## Overview

Store and manage your passwords securely. All entries are AES-256 encrypted.

## Features

-   **Add** a new password entry for different modules (module specific site, username, password, notes)
-   **View** saved passwords (masked by default)
-   **Edit** existing entries
-   **Delete** entries
-   **Copy** password to clipboard

## Tips

-   Use the **eye icon** to reveal/hide a password.
-   Passwords are only visible to the logged-in user.
-   Sysadmin cannot access your passwords.
