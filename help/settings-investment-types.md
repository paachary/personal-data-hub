# Settings — Investment Types Help

> 🔐 Sysadmin only

## Overview

Manage the investment modes available when adding investments.

## Default Investment Types

| Code    | Description                |
| ------- | -------------------------- |
| LUMPSUM | Lumpsum / One Time         |
| SIP     | Systematic Investment Plan |
| SWP     | Systematic Withdrawal Plan |
| STP     | Systematic Transfer Plan   |

## Features

-   **Add** new investment types
-   **Edit** existing types
-   **Delete** unused types
