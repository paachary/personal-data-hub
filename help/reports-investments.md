# Investment Reports Help

> 🔐 Sysadmin only

## Overview

View a consolidated report of all investments across all users.

## Features

-   View **all users' investments** in one place
-   Filter by **instrument type**, **user**, **date range**
-   See **total invested amount** per user and overall
-   Track **active vs closed** investments
-   View **upcoming maturities**

## Tips

-   Use this to get a full picture of the family's or organisation's investment portfolio.
-   Closed investments are shown separately.
