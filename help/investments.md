# Investments Help

## Overview

Track all your investments linked to your bank accounts.

## Features

-   **Add** an investment (name, amount, instrument type, investment type, dates)
-   **Edit** existing investments
-   **Close** an investment when matured
-   **Delete** investments
-   View all investments in a table

## Fields

| Field           | Description                  |
| --------------- | ---------------------------- |
| Investment Name | Label for the investment     |
| Amount          | Invested amount              |
| Instrument Type | FD, MF, LIC, NPS, etc.       |
| Investment Type | Lumpsum, SIP, SWP, STP       |
| Investment Date | Date of investment           |
| Maturity Date   | Expected maturity (optional) |
| Bank Account    | Linked bank account          |

## Tips

-   Closed investments are still visible but marked as closed.
-   Sysadmin can see all users' investments in the Reports section.
