# Admin — Users Help

> 🔐 Sysadmin only

## Overview

Manage all user accounts in the application.

## Features

-   **Create** new user accounts
-   **Reset** a user's password
-   **Reset** a user's MFA
-   **View** all registered users

## Tips

-   When you reset a password, the user will be forced to change it on next login.
-   When you reset MFA, the user will be asked to set up MFA again on next login.
-   You cannot delete user accounts (to preserve data integrity).
