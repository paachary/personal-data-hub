# Dashboard Help

## Overview

The Dashboard gives you a quick summary of all your personal data.

## What You'll See

| Card        | Description                 |
| ----------- | --------------------------- |
| Passwords   | Total saved vault entries   |
| Notes       | Total saved notes           |
| Todos       | Number of **pending** todos |
| Banks       | Total bank records          |
| Investments | Total investment records    |

## Tips

-   Click any section from the **sidebar** to navigate to it.
-   Counts update every time you open the dashboard.
