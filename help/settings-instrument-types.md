# Settings — Instrument Types Help

> 🔐 Sysadmin only

## Overview

Manage the types of financial instruments available when adding investments.

## Default Instrument Types

| Code  | Description                  |
| ----- | ---------------------------- |
| FD    | Fixed Deposit                |
| MF    | Mutual Funds                 |
| LIC   | Life Insurance               |
| NPS   | National Pension Scheme      |
| MI    | Medical Insurance            |
| ELSS  | Equity Linked Savings Scheme |
| PPF   | Public Provident Fund        |
| BONDS | Bonds                        |

## Features

-   **Add** new instrument types
-   **Edit** existing types
-   **Delete** unused types
